# Semana 01 - Calculadora de Sumas

## Descripcion

Este programa en Python lee las lineas desde la entrada estandar, cada linea contine valores seprados por comas.

El programa limpia caracteres invalidos, conviertee los avlores a numeros y calcula la suma de esos valores.

## Como ejecutar

Al ejecutar el programa desde la terminal:
python3 maib.py

De igual forma se puede ejecutar usando el archivo de entrada:
python3 main.py < entrada1.txt
