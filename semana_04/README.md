# Sistema de Inventario Modular

# Descripcion
Este proyecto implementa un sistema de inventario en Python con arquitectura modular.
Permite leer productos desde un archivo CSV, validar la información, identificar productos que necesitan reabastecimiento y generar un reporte automatizado.
